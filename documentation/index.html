Работа с белым цветом

Пиксель белого цвета используется для печати первого значения стека в консоль как число

На фотографии:

  • Блок 1: добавляем в стек 2 и 3

    Результат: стек <2, 3>

  • Блок 2: печатаем значение первого значения стека в консоль

    Результат: стек <2, 3>, консоль: 3


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(2)
stack.append(3)

print(stack[-1])



Работа с серым цветом

Пиксель серого цвета используется для печати первого значения стека в консоль как символ
При таком методе печати все числа автоматически переводятся в символы используя ACII

На фотографии:

  • Блок 1:

    1. Добавляем в стек 4, 4, 4
    2. Умножаем первые два значения 3 раза, т.е. 4x4x4
    3. Добавляем в стек 8
    4. Складываем первые два значения

    Результат: стек <72>

  • Блок 2: печатаем значение первого значения стека в консоль как символ

    Результат: стек <72>, консоль: H


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(4)
stack.append(4)
stack.append(4)

temp = stack[-1]*stack[-2]
stack = stack[:len(stack)-2]
stack.append(temp)

temp = stack[-1]*stack[-2]
stack = stack[:len(stack)-2]
stack.append(temp)

stack.append(8)

temp = stack[-1]+stack[-2]
stack = stack[:len(stack)-2]
stack.append(temp)

print(chr(stack[-1]))



Работа с черным цветом

Пиксель черного цвета используется для обозначения начала и конца блока кода
При этом блок блок кода может быть открыт на одной строке, а закрыт на любой следующей
Блоки кода выполняются последовательно, если не были экранированы
Нумерация первого блока начинается с единицы и увеличивается с каждым последовательным блоком на единицу

На фотографии:

  • Блок 1: добавляем в стек 3

    Результат: стек <3>

  • Блок 2: добавляем в стек 4

    Результат: стек <3, 4>

  • Блок 3: печатаем значение первого элемента в консоль

    Результат: стек <3, 4>, консоль: 4


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(3)

stack.append(4)

print(stack[-1])



Работа с желтым цветом

Пиксель желтого цвета используется для сложения первых двух чисел стека
Этот пиксель извлекает первые два числа из стека, складывает их, а затем помещает результат в массив
Важно учитывать, что числа удаляются из стека и заменяются результатом сложения

На фотографии:

  • Блок 1:

    1. Добавляем в стек 2 и 3
    2. Складываем первые два числа из стека, т.е. 2+3
    3. Печатаем значение первого элемента в консоль

    Результат: стек <5>, консоль: 5


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(2)
stack.append(3)

temp = stack[-2] + stack[-1]
stack = stack[:len(stack)-2]
stack.append(temp)

print(stack[-1])



Работа с зеленым цветом

Пиксель зеленого цвета используется для добавления в стек значения счетчика
При добавлении счетчик обнуляется

На фотографии:

  • Блок 1: добавляем в стек 2, 0 и 3

    Результат: стек <2, 0, 3>


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(2)
stack.append(0)
stack.append(3)



Работа с оранжевым цветом

Пиксель оранжевого цвета используется для вызова блока кода который соответствует значению счетчика
При вызове блока кода счетчик обнуляется

На фотографии:

  • Блок 1: добавляем в стек 4

    Результат: стек <4>

  • Блок 2: увеличиваем счетчик на один и вызываем первый блок кода (2 раза)

    Результат: стек <4, 4, 4>


Приведем пример этой же программы на языке программирования Python:

stack = []

def first():
    stack.append(4)
first()

first()
first()



Работа с синим цветом

Пиксель синего цвета используется для приема пользовательского ввода
Введенные данные автоматически добавляются в стек
Важно помнить, что в случае если пользователь вводит символ или нецелое число, то оно переводится в целые числа используя ACII
Также стоит отметить, что в случае если пользователь вводит несколько символов, то в стек добавляется несколько значений - код каждого символа
Следует знать, что в интерпретаторе ввод выглядит как два знака больше (>>)

На фотографии:

  • Блок 1:

    1. Пользователь вводит число
    2. Печатаем значение первого элемента в консоль

    Результат: стек <(пользовательское число)>, консоль: (пользовательское число)


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(int(input())) # При условии что вводится точно целое число

print(stack[-1])



Работа с циановым цветом

Пиксель цианового цвета используется для создания цикла, который будет повторяться столько раз, сколько установлено на счетчике
При создании цикла счетчик обнуляется
При работе с циановым цветом важно не забывать, что цикл может быть завершен в том же блоке парным пикселем такого же цвета

На фотографии:

  • Блок 1:

    1. Увеличиваем счетчик на 4
    2. Определяем цикл из 4 повторений
    3. На каждой итерации добавляем в стек 2
    4. Закрываем цикл парным пикселем чтобы цикл не затрагивал следующие операции этого блока кода
    5. Увеличиваем счетчик на 3
    6. Определяем цикл из 3 повторений
    7. На каждой итерации печатаем первое значение стека и умножаем первые два значения

    Результат: стек <8>, консоль: 2, 4, 8


Приведем пример этой же программы на языке программирования Python:

stack = []
counter = 0

counter += 4

for i in range(counter):
    stack.append(2)
counter = 0

counter += 3
for i in range(counter):
    print(stack[-1])

    temp = stack[-2] * stack[-1]
    stack = stack[:len(stack)-2]
    stack.append(temp)



Работа с красным цветом

Пиксель красного цвета используется для увеличения значения счетчика на один
Счетчик - независимая переменная с которой взаимодействуют некоторые другие цвета

На фотографии:

  • Блок 1:

    1. Увеличиваем счетчик на 2
    2. Добавляем в стек 2
    3. Печатаем значение первого элемента в консоль

    Результат: стек <2>, консоль: 2


Приведем пример этой же программы на языке программирования Python:

stack = []
counter = 0

counter += 2
stack.append(counter)
counter = 0

print(stack[-1])



Работа с цветом хаки*

Пиксель цвета хаки используется для экранирования блока кода
Если пиксель этого цвета находится в блоке кода, то этот блок не выполняется без вызова
Важно знать, что такой пиксель может находиться в любом месте внутри блока кода для его экранирования

На фотографии:

  • Блок 1 (не выполняется): добавляем в стек 4

    Результат: в стек добавляется 4

  • Блок 2: увеличиваем счетчик на один и вызываем первый блок кода (2 раза)

    Результат: стек <4, 4>


Приведем пример этой же программы на языке программирования Python:

stack = []

def first():
    stack.append(4)

first()
first()

*См. разницу с работой с оранжевым цветом




Работа с розовым цветом

Пиксель розового цвета используется для того, чтобы разделить второе число стека на первое
Этот пиксель извлекает первые два числа из стека, делит предпоследнее на первое, а затем помещает результат в массив
Важно учитывать, что числа удаляются из стека и заменяются результатом деления
Также важно учитывать, что результат деления округляется по правилам округления

На фотографии:

  • Блок 1:

    1. Добавляем в стек 4 и 2
    2. Делим 4 на 2

    Результат: стек <2>


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(4)
stack.append(2)

temp = int(stack[-2]/stack[-1])
stack = stack[:len(stack)-2]
stack.append(temp)



Работа с темно-желтым цветом

Пиксель темно-желтого цвета используется для того, чтобы умножить первое число стека на второе
Этот пиксель извлекает первые два числа из стека, умножает их, а затем помещает результат в массив
Важно учитывать, что числа удаляются из стека и заменяются результатом умножения

На фотографии:

  • Блок 1:

    1. Добавляем в стек 4 и 2
    2. Умножаем 4 на 2

    Результат: стек <8>


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(4)
stack.append(2)

temp = stack[-2]*stack[-1]
stack = stack[:len(stack)-2]
stack.append(temp)

Работа с темно-зеленым цветом

Пиксель темно-зеленого цвета используется для создания условия, которое проверяет равно ли значение первого элемента стека значению счетчика
При создании условия счетчик обнуляется, даже если условие не будет выполняться
При работе с этим цветом важно не забывать что условие может быть завершено в том же блоке парным пикселем такого же цвета

На фотографии:

  • Блок 1:

    1. Добавляем в стек 2, 2

    Результат: стек <2, 2>

  • Блок 2:

    1. Пользователь вводит число
    2. Увеличиваем значение счетчика на 1
    3. Создаем условие: если первый элемент массива равен 1, то
    4. Если условие выполняется то: удаляем первый элемент стека
    5. Если условие выполняется то: складываем первые два числа стека
    6. Если условие выполняется то: печатаем первое число стека
    7. Закрываем условие парным пикселем чтобы условие не затрагивало следующие операции этого блока кода
    8. Увеличиваем значение счетчика на 2
    9. Создаем условие: если первый элемент массива равен 2, то
    10. Если условие выполняется то: удаляем первый элемент стека
    11. Если условие выполняется то: умножаем первые два числа стека
    12. Если условие выполняется то: печатаем первое число стека

    Результат: в зависимости от введенного значения


Приведем пример этой же программы на языке программирования Python:

stack = []
counter = 0

stack.append(2)
stack.append(2)

stack.append(int(input())) # При условии что вводится точно целое число

counter += 1

if counter == stack[-1]:
    del stack[-1]

    temp = stack[-2] + stack[-1]
    stack = stack[:len(stack)-2]
    stack.append(temp)

    print(stack[-1])
counter = 0

counter += 1
counter += 1

if counter == stack[-1]:
    del stack[-1]

    temp = stack[-2] * stack[-1]
    stack = stack[:len(stack)-2]
    stack.append(temp)

    print(stack[-1])

Работа с темно-красным цветом

Пиксель темно-красного цвета используется для того, чтобы изменить знак первого числа стека на противоположный

На фотографии:

  • Блок 1:

    1. Добавляем в стек 4, 2
    2. Изменяем знак первого элемента стека

Работа с темно-розовым цветом

Пиксель темно-розового цвета используется для того, чтобы переместить последний элемент стека на первое место
Важно понимать, что при этом стек сдвигается на один элемент (первый элемент становится вторым, второй - третьим, и т.д.)

На фотографии:

  • Блок 1:

    1. Добавляем в стек 3, 1
    2. Переносим последний элемент в начало стека

    Результат: стек <1, 3>


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(3)
stack.append(1)

temp = stack[0]
del stack[0]

stack.append(temp)



Работа без цвета

Цвет не используется когда никакие операции выполнять не нужно
Важно знать, что пиксели без цвета полностью игнорируются и могут служить визуальным разделением элементов блока кода

На фотографии:

  • Блок 1:

    1. Добавляем в стек 2, 3
    2. Складываем первые два значения стека
    3. Печатаем первый элемент стека в консоль

    Результат: стек <5>, консоль: 5


Приведем пример этой же программы на языке программирования Python:

stack = []

stack.append(2)
stack.append(3)

temp = stack[-2] + stack[-1]
stack = stack[:len(stack)-2]
stack.append(temp)



Примеры и описание программ

  1. «Hello, world!»

    2. Так может выглядеть ваша первая программа на моем языке

    Страшно? Я уверен, что да

    Теперь по порядку:

    1. Для того чтобы напечатать в консоль «Hello, world!» нам необходимо набрать ACII код каждой буквы
    2. Для этого в каждой строке необходимо использовать математические операции для быстрого набора числа
    3. Обратимся к одному из инструментов ниже для перевода фразы «Hello, world!» в ACII код
      Получаем: <72 101 108 108 111 44 32 119 111 114 108 100 33>
      Это должен быть наш конечный стек
    4. Теперь, вооружившись знаниями математики, заполняем стек нужными значениями
    5. Блок кода 1:
      Добавляем в стек 4, 4, 4
      Перемножаем это и получаем 64
      Добавляем в стек 8 и складываем с 64
      В итоге получаем стек <72>
    6. Блок кода 2:
      Добавляем в стек 5, 4, 5
      Перемножаем это и получаем 100
      Добавляем в стек 1 и складываем с 100

    7. Дальше можно действовать по похожей схеме и набирать нужные числа
